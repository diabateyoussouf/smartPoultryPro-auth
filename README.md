## Auth Service

    Ce microservice gère l’authentification des utilisateurs dans l’architecture SmartFlocks.
    Il permet l’inscription, la connexion et la génération de tokens JWT utilisés par les autres services via l’API Gateway.

## Fonctionnalités

    Inscription d’un utilisateur
    
    Connexion avec email et mot de passe
    
    Génération d’un token JWT
    
    Gestion des rôles (ex : USER, ADMIN)
    
    Stockage des utilisateurs dans MySQL

## Technologies

    Java 21
    
    Spring Boot 3
    
    Spring Security
    
    JPA / Hibernate
    
    MySQL 8
    
    Docker & Docker Compose
